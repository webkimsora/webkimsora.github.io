    스와이퍼 옵션 사용하기

    스와이퍼의 옵션을 swiperOptions에서 동일하게 사용할 수 있습니다.

    • 

        
    
        • 
    
        • 
    
        • 

      


<script>
$('.example_slider').cnSlider({
    swiperOptions: {
        loop: true,
        autoplay: {
            delay: 3000,
            disableOnInteraction: false,
        },
    }
})
</script>

    커스텀 옵션 사용하기

    html 추가가 필요한 navigation(controls), pagination(pager), thumbs(nav) 옵션은 따로 옵션을 만들어 쉽게 추가할 수 있습니다.
    pagerClassName, controlsClassName를 사용하여 클래스이름을 변경할 수 있고, thumbOptions과 navOptions을 사용하여 등록된 thumbs옵션과 등록된 슬라이드의 옵션을 설정할 수 있습니다.

      • 슬라이드 1
      • 슬라이드 2
      • 슬라이드 3
    • 

        
    
        • 
    
        • 
    
        • 

      


<script>
$('.example_slider').cnSlider({
    // pagination 설정과 생성되는 class 이름 설정
    pager: true,
    pagerClassName: 'example_pagination',

    // navigation 설정과 생성되는 class 이름 설정
    controls: true,
    controlsClassName: 'ico ico_arrow_left example_control example-control-',

    // thumbs 설정과 옵션
    nav: '.slide_nav',
    thumbOptions: { ... }

    // thumb로 등록된 슬라이드 옵션
    navOptions: { ... }

    // 스와이퍼 옵션
    swiperOptions: {
        loop: true,
        autoplay: {
            delay: 3000,
            disableOnInteraction: false,
        },
    }
})
</script>
  • Name Type Default Description
    swiperOptions objects {}

    기존 swiper의 옵션을 설정합니다. 옵션은 여기에서 확인 가능합니다. 

    $('#ex_slider').cnSlider({
    swiperOptions: {
        // swiper options
        loop: true,
        autoplay: {
            delay: 5000,
        },
    }
})
    pager boolean false

    페이징 사용 여부를 설정합니다.
    pagerClassName string -

    페이징 버튼에 커스텀 클래스를 추가할 수 있습니다.
    controls boolean false

    컨트롤 사용 여부를 설정합니다.
    controlsClassName string [className]-prev,
    [className]-next,

    컨트롤 버튼에 커스텀 클래스를 추가할 수 있습니다.
    nav string -

    스와이퍼의 thumbs속성을 클래스명만 입력하여 간단하게 설정할 수 있습니다.
    navOptions objects {}

    nav에 등록된 슬라이더의 속성을 여기에서 추가할 수 있습니다. 

    $('#ex_slider').cnSlider({
    nav: '.ex_nav',
    navOptions: {
        // .ex_nav options
        slidesPerView: 'auto'
    }
})
    thumbOptions objects {}

    스와이퍼의 thumbs 속성을 여기에서 추가할 수 있습니다. 

    $('#ex_slider').cnSlider({
    nav: '.ex_nav',
    thumbOptions: {
        // .ex_nav thumb options
        slideThumbActiveClass: 'active',
        autoScrollOffset: 2
    }
})
    returnAll boolean false

    nav를 사용할 경우 true로 설정하면 nav슬라이드도 리턴됩니다.
    네비게이션 슬라이드를 제어해야할 때 사용할 수 있습니다.

    const swiper = $('#ex_slider').cnSlider({
    nav: '.ex_nav',
    swiperOptions: {...}
})

let slide = swiper.slide; // 기존 슬라이드
let slide = swiper.slide_nav; // 네이게이션 슬라이드

// ex) 네이게이션 슬라이드 이동하기
swiper.slide_nav.slideTo(2, 0);

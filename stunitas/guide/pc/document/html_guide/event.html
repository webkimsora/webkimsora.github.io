클래스 규칙

  1. 이벤트 페이지 시작시 클래스에 pc지라주소, 모바일 경우 pc지라주소_m 클래스로 각 고유성 확보
    /* PC */

    
/* Mobile */
  2. .inner-wrap 클래스는 --layout-width 변수에 정의된 통 inner 일때 사용하고
    페이지 내에서는 .inner 로 기존과 동일하게 사용 바랍니다.
  3. layout을 잡을땐 .inner 를 사용합니다.
  4. 퀵 배너는 position:sticky 를 사용 해야 하므로 전체 overflow:hidden; 은 사용하지 않습니다.
    overflow:hidden; 가 필요하면 각 단에서 사용바랍니다.
    특히, swiper 작업 후 확인 바랍니다.

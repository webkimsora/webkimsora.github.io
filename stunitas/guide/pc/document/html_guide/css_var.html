CSS변수

커넥츠 HTML/CSS 문서

복잡한 웹사이트 같은 경우 엄청난 양의 CSS를 가지고 있는데, 유지보수를 하다 보면 여러 곳에서 사용한 임의의 값을 한꺼번에 바꿔야 할 때가 있습니다.
그 값들을 일일이 바꾸려면 상당히 복잡하게 되므로, 여타 프로그래밍 언어처럼 변수를 이용하려 합니다.
요소 전반에 걸쳐 사용되는 임의의 값을 변수에 저장하고 호출하면 보다 효율적인 프로그래밍이 가능해집니다.
css변수 사용
css변수 네이밍은 전체 소문자로 사용합니다.
:root{}는 일종의 전역변수같은 느낌으로, HTML 문서 어디에서나 사용자 지정 속성에 접근할 수 있도록 구성합니다.
element{}는 지역변수와 같아 해당 엘리먼트에만 css변수처리 가능합니다.
css변수는 컴포넌트에 사용을 위함으로, 특수한 상황이 아닐 경우에는 :root에서 css변수를 만들도록 합니다. 
:root { //변수지정
    --main-bg-color: brown;
}

element {background: var(--main-bg-color);}
CSS변수 네이밍
  1. 기본을 명시할때는 primary로 지정해줍니다.
  2. 폰트사이즈는 변수명을 text로 표기합니다.
    13px이 공통 기본 사이즈로 기본은     primary로 지정되었습니다.
    디자인에서 폰트 기준이 제목(Title)과 본문(Body)이 있습니다만, 사이즈는 고정이기에 전체 12px, 13px, 14px, 16px, 20px를 변수처리 되었습니다.
  3. 폰트 컬러는 영어로 첫 문자를 축약합니다.
    기본은 blue가 주된 사용이지만 각 단기별로 각각의 고유 컬러가 나뉘어져 있습니다.

  4. CSS변수 BASIC

    Name Type Default Description
    layout .inner --layout 컨텐츠 레이아웃을 잡을 때 사용합니다.
    font-size title
    body    		 --text-(el)

    * primary: basic 사이즈
    기본 사이즈인 13px은 text-primary, 타이틀에서만 쓰이는 20px은 text-title, 그 외 고정된 12px, 14px, 16px는 px 숫자를 적는다.

    ex)--text-12: 12px;, --text-14: 14px;

    font-weight --weight-(el)

    폰트 굵기는 400,500,700,900 사용

    --weight-extra: 900;
    --weight-bold: 700;
    --weight-mid: 500;
    --weight-primary: 400;

    color --color값-(el)

    색상은 color 표기후 색상 영문의 앞문자를 사용
    동일 컬러 리스트는 최대 5가지로 단위는 20단위로 붙여줍니다.
    하지만 grayscale은 총 10가지이므로 10~100까지 10단위를 사용합니다.

    ex) blue 색상 3번째 -> --blue-60

    .sobang__wrap {color: var(--red-20);}
.content_box {color: var(--gray-30);}

:root {
    --color-primary: #121212; /* 기본 color */
    --color-secondary: #7E8B9A; /* 보조 color */
    --color-disabled: #C5CBD3; /* 비활성화 color */
    --color-highlight: #025AF6; /* 강조 color */
}

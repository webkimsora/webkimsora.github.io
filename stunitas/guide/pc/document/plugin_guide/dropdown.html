드롭다운 열어두기

메뉴 클릭시 드롭다운을 닫히지 않게 설정합니다.

  • 드롭메뉴1
  • 드롭메뉴2
  • 드롭메뉴3
$('#btn').cnDropdown({
    closeOutside: false,
    autoMenuClose: false,
})

Options

Name Type Default Description
width number 94

드롭가운의 가로 사이즈를 설정합니다.
closeOutside boolean true

바깥 영역 클릭시 드롭다운의 닫힘 여부를 정합니다.
autoMenuClose boolean true

이 설정을 끄면 드롭메뉴 버튼 클릭시 드롭메뉴가 자동으로 사라지지 않습니다.
offset object {x: 2, y: 4}

드롭다운 생성시 버튼과의 여백을 조정합니다.
onMenuClick function -

드롭다운 버튼 클릭시 이벤트를 실행하며, 각 버튼의 인덱스를 반환합니다.
pEl : 현재 드롭다운메뉴
pIndex : 클릭한 드롭다운메뉴 인덱스
pParent : 드롭다운메뉴를 펼치기 위해 클릭한 버튼(여기에 각 아이템 고유 속성 data-로 지정해서 접근 가능)

$('.modal').cnDropdown({
    onMenuClick: function(pEl, pIndex, pParent){
        console.log(pIndex);
        console.log('개발에서 쓸 데이터 각 버튼', pParent.data('test') );
    },
})

기본 사용법

스크롤 이벤트나 애니메이션을 추가합니다. 스크롤 되는 엘리먼트를 지정하여 스크립트를 적용시키고, 애니메이션 대상에게 클래스 추가 후에 targetClass로 지정합니다.
targetClass로 지정한 엘리먼트는 각 위치에 도달했을 때 active클래스가 추가됩니다. 

$(window).cnScroll({
    targetClass: 'scrollAni'
});

Options

Name Type Default Description
targetClass string -

스크롤시 이벤트를 적용할 엘리먼트의 클래스를 작성합니다.
once boolean false

활성화시 스크롤 트리거를 처음 한번만 실행합니다.

$('.nav_type2').cnTab({
    once: true
})

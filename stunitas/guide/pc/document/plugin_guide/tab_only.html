Options

Name Type Default Description
width string 'auto', '100px', '50%', 'css'

각 버튼의 너비를 설정합니다. (단위와 함께 입력 예: '100px')
이 옵션을 넣지 않으면 탭 갯수에 따라 자동으로 너비가 조정됩니다.
'css' 옵션은 밑에 사용자가 설정한 css의 넓이 및 여백을 사용합니다.
'css' 로 설정하면 gap 옵션은 무시 됩니다.
탭은 display:flex 로 되어 있습니다.
spaceTab number n

각 탭 사이의 공백 숫자로 입력해야 하며 뒤에 px이 자동 붙습니다.
기본 - 설정안함
startIndex number n

시작 탭 설정(제로베이스)
기본 - 0

activeClass string 'active'

활성화된 버튼과 탭 영역에 추가되는 클래스를 변경합니다.
onClick function -

버튼을 클릭했을 때 이벤트를 추가하며 클릭한 엘리먼트와 인덱스를 반환합니다.
이미 활성화된 버튼을 다시 클릭할 경우 중복해서 실행되지 않습니다. 

$('.custom_nav').cnTab({
    onClick: function(el, index){
        console.log(index + '번째 메뉴 클릭');
        el.addClass('clicked'); // 클릭한 영역에 클래스 추가
    };
});
onInit function -

맨 처음에 로드되고 실행 되는 함수 

$('.custom_nav').cnTab({
    onInit: function(el, index){
        console.log('로드 되고 처음으로 시작할때');
    };
});

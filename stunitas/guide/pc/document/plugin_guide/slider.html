컨트롤과 함께 사용하기

컨트롤을 추가하려면 따로 html을 추가할 필요 없이 옵션에서 활성화만 하면 자동으로 추가됩니다. pager: true와 controls: true 옵션을 이용해보세요.
컨트롤에 클래스를 추가해야할 경우 pagerClassName과 controlsClassName 설정으로 추가할 수 있습니다.

슬라이드 1
슬라이드 2
슬라이드 3
$ ('#ex_slider').cnSlider({
    pager: true,
    pagerClassName: 'custom-pager', 
    controls: true, 
    controlsClassName: 'custom-control' // 각각 [custom-control]-prev와 [custom-control]-next가 추가됨
});

    Options

    Name Type Default Description
    swiperOptions objects {}

    기존 swiper의 옵션을 설정합니다. 옵션은 여기에서 확인 가능합니다. 

    $('#ex_slider').cnSlider({
    swiperOptions: {
        // swiper options
        autoplay: {
            delay: 5000,
        },
        loop: true,
    }
})
    pager boolean false 페이징 사용 여부를 설정합니다.
    pagerClassName string - 페이징 버튼에 커스텀 클래스를 추가할 수 있습니다.
    controls boolean false 컨트롤 사용 여부를 설정합니다.
    controlsClassName string [className]-prev,
    [className]-next,     		컨트롤 버튼에 커스텀 클래스를 추가할 수 있습니다.
    nav string -

    스와이퍼의 thumbs속성을 클래스명만 입력하여 간단하게 설정할 수 있습니다.
    navOptions objects {}

    스와이퍼의 thumbs 속성을 여기에서 추가할 수 있습니다. 

    $('#ex_slider').cnSlider({
    nav: '.ex_nav',
    navOptions: {
        // nav(thumbs) options
        slidesPerView: 'auto'
    }
})

기본 사용법

이 플러그인에선 position: fixed를 사용하지 않고 position: sticky를 사용합니다.
네비게이션 버튼이나 추가 옵션이 필요하지 않은 단순 배너의 경우 스크립트가 아닌 css만 사용하여 적용합니다. 


.top_bar{position: sticky; top: 0;}


.top_bar{position: sticky; bottom: 0;}

CSS 사용하기

추가 이벤트가 필요하지 않은 경우 CSS만 활용하여 적용합니다.
이 경우 고정될 배너의 위치는 최상위 혹은 특정 영역이 지정되지 않은 곳에 위치하여야 합니다.

Options

Name Type Default Description
mode string 'top'

배너의 위치를 설정합니다. 하단 고정 배너 내부에 버튼이 있을 경우, 이 옵션을 'bottom'으로 변경해야 정상 작동합니다.

$('.nav_type2').cnTab({
    mode: 'top'
})
target string -

윈도우 스크롤이 아닌 커스텀 스크롤 박스 내에 배너를 삽입할 경우 이 설정으로 지정합니다.
activeClass string 'active'

배너가 고정되었을 때 배너에 추가되는 클래스명을 변경합니다.
data-target string -

배너 안에 버튼에게 영역별 스크롤 클래스를 추가하기 위해 해당 영역을 지정합니다.

progress boolean false

이 옵션을 활성화하면 네비게이션 버튼에 설정한 영역(target)에서 벗어나도 활성화 클래스가 사라지지 않습니다.
onScroll function -

스크롤했을 때 이벤트를 추가합니다. 첫번째 인자로는 적용된 스크롤 배너의 엘리먼트를 반환하며,
두번째 인자로는 상단 고정 여부를 boolean 타입으로 리턴합니다.

$('.scrollEl').cnStickyBar({
    onClick: function(active){
        if(el, active){
            console.log('상단 고정');
        }
    };
});

<script type="text/javascript">
    $(function(){
        // scroll 영역 모션
        var controller = new ScrollMagic.Controller();

        var wipeAnimation = gsap.timeline()
            .add("step1")
            .fromTo(".content-panel-1", 1, { scale: 1, y: 0 }, { scale: 0.95, y: -10 }, "step1")
            .fromTo(".content-panel-2", 1, { y: "300%" }, { y: "10px" }, "step1")

            .add("step2")
            .fromTo(".section-group-1", 1, { scale: 1, y: 0 }, { scale: 0.95, y: -10 }, "step2")
            .fromTo(".content-panel-3", 1, { y: "300%" }, { y: "10px" }, "step2")

            .add("step3")
            .fromTo(".section-group-2", 1, { scale: 1, y: 0 }, { scale: 0.95, y: -10 }, "step3")
            .fromTo(".content-panel-4", 1, { y: "300%" }, { y: "10px" }, "step3");

        var scene = new ScrollMagic.Scene({
            triggerElement: ".cont_scroll",
            triggerHook: 0.1, 
            duration: "300%"
        })

        .setPin(".cont_scroll")
        .setTween(wipeAnimation) 
        .addTo(controller);
    })
</script>

    📌 Trigger 관련 옵션

    기본적으로 trigger은 순차적으로 적용됩니다. add를 사용하여 동시에 실행이 가능합니다. add는 gsap 메서드 뒤에 함께 추가하여 사용가능합니다.

    GSAP 애니메이션 메서드

    • to() → 현재 상태에서 지정한 값으로 이동
    • from() → 특정 값에서 현재 상태로 이동
    • fromTo() → 특정 시작 값에서 특정 끝 값으로 이동
    옵션 역할
    triggerElement 스크롤이 도달하면 애니메이션을 시작할 트리거 요소를 설정
    triggerHook: 0 화면 상단에서 효과 발생
    triggerHook: 0.5 화면 중앙에서 효과 발생
    triggerHook: 1 화면 하단에서 효과 발생
    triggerHook: "onLeave" 요소가 화면에서 사라질 때 효과 발생
    triggerHook: "onEnter" 요소가 화면에 등장할 때 효과 발생
    duration 애니메이션이 지속될 스크롤 거리(시간) 설정 (예: 300% → 스크롤 길이 3배 지속)
    🎬 애니메이션 & 동작 관련 옵션
    옵션 역할
    setPin 요소를 스크롤 중 고정시켜 배경처럼 보이게 함
    setTween GSAP 또는 TweenMax 애니메이션을 적용
    reverse: true 스크롤을 되감으면 애니메이션이 다시 실행됨 (기본값: true)
    reverse: false 한 번만 실행하고 다시 되감아도 애니메이션이 반복되지 않음

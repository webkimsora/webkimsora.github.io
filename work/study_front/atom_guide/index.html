atom설치

Atom의 특징
  • 무료이다.
    많은 사용자들을 보유하고 있는 Sublime Text, Edit Plus 등과는 반대로 무료로 지원한다는 점
  • 플러그인으로 기능을 확장할 수 있다.
  • HTML / CSS /JavaScript와 같은 웹기술들로 구현되었다.
  • Windows / MAC / Linux 모두 설치 가능하다.
node.js 설치
  • 아톰에디터가 아니더래도 작업시 항상필요하니 필수적으로 설치 (https://nodejs.org/ko/)
  • 안정화 버전 선호
  • 경로는 항상 기본으로(설치시 지정되어있음)
Shortcut Key
Editor
  • Ctrl + Shift + P: 커맨드 팔레트 열기
  • Ctrl + ,: 설정 뷰
  • Ctrl + .: 키 바인딩 보기 (단축키 작동 현황 보기)
  • Ctrl + \: Treeview 열기/닫기
  • Ctrl + Alt + I: 개발자 콘솔 열기
Tree View (활성화 상태일 때)
  • A: 새로운 파일 생성
  • Shift + A: 새로운 폴더 생성
  • M or F2: 파일(폴더) 경로 및 이름 변경
  • Delete or Backspace: 파일(폴더) 삭제
  • D: 파일(폴더) 복사
Tab
  • Ctrl + T: 파일 파인더 열기 (탭 전환)
  • Ctrl + Tab: 다음 탭 전환
  • Ctrl + Shift + Tab: 이전 탭 전환
  • Crtl + Shift + T: 최근에 닫은 탭 다시 열기
General Editing
  • Ctrl + /: 주석 토글
  • Ctrl + F: 찾기/바꾸기
  • Ctrl + Shift + F: 프로젝트 전체에서 찾기
  • Crtl + Shift + T: 최근에 닫은 탭 다시 열기
  • Ctrl + G: 라인 번호로 커서 이동
  • Ctrl + R: 키워드로 이동
  • Ctrl + M: 블럭 매칭
  • Ctrl + J: 라인 조인
  • Ctrl + L: 라인 선택
  • Ctrl + Shift + K: 현재 라인 삭제
  • Ctrl + Shift + D: 현재 라인 다음 라인으로 복사
  • Ctrl + ↑, ↓: 현재 라인 이동
  • Ctrl + Alt + ↑, ↓: 다중 커서 삽입
  • Ctrl + Space: 코드 힌트 보기
  • Shift + 방향키: 텍스트 선택
setting
  • 단축키 'Ctrl + ,' -> editor
  • 'Tab Length' = 4
  • 탭사이즈 외에는 기본 셋팅값 사용권장
whitespace
  • 단축키 'Ctrl + ,' -> packages -> Core packages
  • whitespace 패키지 자체를 비활성화
  • 파일 저장시 마지막라인의 공란 단어와 단어사이의 공란 등등
    작업자가 의도하지 않게 저장되는 경우가 생기기 때문에 비활성화한다.
style.less
  • 아톰에디터는 사용자에 맞게 에디트자체 커스텀이 가능하다.
  • File > stylesheet
Theme Setting
  • Ctrl+, > Theme 메뉴에서 테마설정이 가능합니다.
  • 추가로 검색해보면 수 많은 테마가 있고 마음에 드는 테마를 설치하여 사용할 수 있습니다.
패키지설치
  • 단축키 'Ctrl + ,' -> install
    Ftp-remote-edit

    - 초반 셋팅시 개인사용자 비밀번호를 설정하여야합니다.
    - 이후 ftp-remote-edit 활성화시 비밀번호를 치고 들어가야합니다.
    - 관련ftp/sftp 정보를 입력후 save 버튼을 눌러 사용합니다.

    Atom-beautify* PHP파일은 정렬불가능

    - 코드를 자동정렬 해주는 패키지
    - 언어별로 정렬을 지정할수있다. css는 정렬에서 제외
    - php같은 경우 정렬은 auto-indent 를 사용한다.(하단부에 auto-indent 설명)

    auto-indent

    - 탭 정렬을 자동으로 맞쳐주는 플러그인으로
    - 프로젝트 진행시(특히 버전관리를 할때) auto-indent 후 저장 또는 커밋
    - 단축키 : Ctrl + Shift + i

    Project-manager

    - 프로젝트를 개별적으로 관리해준다.
    - Packages > Project-Manager > Edit Projects
    - 추가 및 수정 이 가능하다.
    - 프로젝트 추가 후 Alt + Shift + P 단축키로 리스트항목 선택하여 프로젝트 진행

    atom-live-sever

    - 톰캣,레진.. 등등 서버 툴 없이 아톰자체적으로 가상서버연결(livereload기능까지포함됨)
    - html파일에서만 사용하는걸 권장
    - 프로젝트 매니저 프로젝트 하나를 툴로 돌릴때 사용하는걸 권장
    - Packages > atom-live-server > Start sever 클릭시 자동으로 브라우저의 포트를 형성시켜준다.

    bracket-mathcher

    - html,css,js 선택부분과 닫는 부분을 표시해준다.

싱크셋팅
  • sync-setting 패키지 설치 후 setting(첫 설치시 오류가 발생함 셋팅값이 없기때문)
  • 깃헙 로그인
  • 신규 토큰 발급 https://github.com/settings/tokens/new
  • 리스트항목 gist 만 체크
  • 발급된 token, id 값을 확인 url (gist.github.com/아이디/토큰 = 식으로 넘어옴)
  • 이제 아톰실행때마다 상태메세지가 뜨고, Backup , Restore 버튼이 있는데 backup은 커밋개념 restore 는 업데이트개념임
  • **주의사항 Token 값은 현재 이 페이지에서만 볼수 있다.
    나중에 보려고 하면 못보고 Regenrate 하거나 새로운 Token을 생성해야한다. 어딘가에 잘 적어두자!!
  • Backup : 현재 설정 정보를 Gist에 백업한다.
  • View backup : Gist로 간다.
  • Restore : Gist의 백업 정보를 불러온다.
  • Dismiss : 아무것도 하지 않고 무시한다.

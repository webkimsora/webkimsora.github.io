GSAP Plugin List

CSS Transition, Jquery Animation으로 HTML 모션을 줄 수 있지만, 작업이 많을경우 소스가 복잡해져 한계가 있다.
또한 속도와 모션 움직임이 끊겨 보일때가 있으므로, 애니메이션 라이브러리인 TWEENMAX의 활용도가 더욱 높아지는 추세다.
특히 SVG, CANVAS에 트윈맥스를 응용하여 모션을 더 부드럽게 처리가 가능하다.
트윈맥스는 greensock 에서 만든 라이브러리로 https://greensock.com/
위 사이트에 접속하여 다운을 받거나 cdn을 연결해주면 사용이 가능하다.

LINK

  • 1. Bezier Plugin

    • - 곡선형 경로를 따라 애니메이션화하고 4가지 방법으로 포인트와 값을 배열로 정의
    • - type 지정: thru, soft, quadratic, cubic
    • - thru 사용시 curviness 메소드를 통해 커브 굴절을 증감할수 있다.
    • - SVG path와 응용해서 사용하기 용이하다.
    • ex) https://codepen.io/jonathan/pen/cqvJe

    LINK

  • 2. CSS Plugin(SVG)

    • - 일반적으로 많이 쓰는 CSS3 애니메이션을 트윈맥스에 적용
    • - ClassName 추가,제거가 가능
    • - SVG 경우 애니메이션 작동 좌표값을 지정하여 중심을 바꿀 수 있다.
    • - CSSRulePlugin을 통해 유사요소(ex.:before, :after)도 객체로 사용 가능

    LINK

    CSSRule LINK

  • 3. draggable Plugin

    • - 마우스를 이용하여 드래그, 회전 등 모션 이벤트 부여할수 있음
    • - 가장 많이 쓰이는 플러그인중 하나
    • - 위치, 회전, 스크롤등으로 드래그 유형 선택가능

      "x,y"| "top,left"| "rotation"| "scroll"| "x"| "y"| "top"| "left"| "scrollTop"|"scrollLeft"

    • - ThrowPropsPlugin(유료)으로 함께 사용 가능
      https://codepen.io/GreenSock/pen/zDwEk

    LINK

  • 4. text Plugin

    • - 텍스트를 배치해주는 효과
    • - 새로 작성 혹은 현재 텍스트와 지정 텍스트를 변환해줄수 있음

    LINK

  • 5. ScrambleText Plugin(유료)

    • - textPlugin과 유사하지만 설정한 값의 텍스트를 무작위 글자와 겹쳐서 보여주는 형태의 효과를 만들수 있다.
    • ex) 매트릭스 효과

    LINK

  • 6. attr Plugin

    • - 숫자 속성을 무제한으로 설정 변경 가능
    • - SVG에서 선(점)모양을 바꿔 사용하기 용이하다.
      SVG points를 배열을 이용해 각 값을 바꿔 이용

    LINK

  • 7. Modifiers Plugin(SVG), MorphSVG plugin

    • - SVG, Ticker 등에서 트윈 기능 값을 도중에 가져와 첫번째 매개변수로 함수에 공급하여 새 값을 반환시키도록 할 수 있다.
    • - 따로 파일 로드를 해줘야한다.

    morphSVG LINK

  • 8. RoundProps Plugin

    • - 모든 속성을 가장 가까운 정수로 반올림
    • - 지정 값때마다 트윈맥스를 실행시켜 나타낼 수 있음
    • - Progress 사용에 용이하다

    LINK
